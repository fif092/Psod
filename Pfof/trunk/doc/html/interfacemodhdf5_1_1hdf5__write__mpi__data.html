Detailed Description

Generic inteface used to write data in a parallel hdf5 file.

Parameters
[in]idid of the file/group where the dataset will be written
[in]namename of the dataset
[in]n1first dimension of the data array
[in]n2second dimension of the data array (optional)
[in]data1-D or 2-D data array of type Integer(4), Integer(8), Real(4) or Real(8)
[in]commMPI communicator used for the output (the same communicator should have been used to create the file)
