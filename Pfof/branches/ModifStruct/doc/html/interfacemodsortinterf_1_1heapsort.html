Detailed Description

Generic inteface used to sort arrays with heapsort algorithm.

Parameters
[in]nLenght of the arrays
[in]trefReference array: this array will be sorted in increasing order, the other arrays will be sorted following this one
[in][tx,tv,tid]Arrays to sort
[in][tp,tf,trid]Optional arrays to sort
