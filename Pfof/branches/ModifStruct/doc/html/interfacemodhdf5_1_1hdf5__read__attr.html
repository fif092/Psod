Detailed Description

Generic inteface used to read attributes from a hdf5 file.

Parameters
[in]idid of the file/group where the attribute will be read
[in]namename of the attribute
[in]n1first dimension of the attribute array (optional)
[in]n2second dimension of the attribute array (optional)
[in,out]data0-D, 1-D or 2-D attribute of type Integer(4), Real(4), Real(8) or Character
