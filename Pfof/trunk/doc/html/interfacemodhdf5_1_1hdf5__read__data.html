Detailed Description

Generic inteface used to read data from a hdf5 file.

Parameters
[in]idid of the file/group where the dataset will be read
[in]namename of the dataset
[in]n1first dimension of the data array
[in]n2second dimension of the data array (optional)
[in,out]data1-D or 2-D data array of type Integer(4), Integer(8), Real(4) or Real(8)
