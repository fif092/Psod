Detailed Description

Generic inteface used to sort arrays with quicksort algorithm.

Parameters
[in]pFirst index of the arrays to sort
[in]rLast index of the arrays to sort
[in]trefReference array: this array will be sorted in increasing order, the other arrays will be sorted following this one
[in][tx,tv,tid]Arrays to sort
[in][tp,tf]Optional arrays to sort
